📋ProPresenter란?

ProPresenter는 라이브 프레젠테이션과 미디어 제작을 위한 전문적인 소프트웨어입니다. 교회, 학교, 기업 등에서 널리 사용되며, 다중 화면 출력, 실시간 편집, 미디어 통합 등의 강력한 기능을 제공합니다.

주요 특징

  • 다중 화면 지원: 최대 8개의 독립적인 출력 화면
  • 실시간 편집: 라이브 중에도 슬라이드 수정 가능
  • 미디어 통합: 비디오, 오디오, 이미지 완벽 지원
  • 원격 제어: 모바일 앱으로 원격 조작
  • 테마 및 템플릿: 전문적인 디자인 요소
💡 초보자 팁: ProPresenter는 14일 무료 체험이 가능합니다. 구매 전에 충분히 테스트해보세요!

💻설치 및 초기 설정

시스템 요구사항

🖥️ macOS

macOS Ventura (13.5) 이상
4GB RAM (8GB 권장)
2GB 여유 공간

🖥️ Windows

Windows 10/11
4GB RAM (8GB 권장)
2GB 여유 공간

설치 과정

  1. 다운로드: 공식 웹사이트에서 최신 버전 다운로드
  2. Mac 설치: ZIP 파일을 압축 해제 후 Applications 폴더로 드래그
  3. Windows 설치: 설치 프로그램 실행 후 안내에 따라 진행
  4. 방화벽 허용: 네트워크 기능 사용을 위해 방화벽 규칙 허용
  5. 첫 실행: 라이선스 동의 및 초기 설정 마법사 완료
⚠️ 주의사항: Mac에서는 데스크탑 및 문서 폴더 접근 권한을 허용해야 합니다. 시스템 환경설정에서 개인정보 보호 설정을 확인하세요.

⚙️기본 기능 익히기

1. 새 프레젠테이션 만들기

  1. 도구 모음에서 "+" 버튼 클릭 또는 Cmd+N (Mac) / Ctrl+N (Windows)
  2. 프레젠테이션 이름 입력
  3. 템플릿 선택 (선택사항)
  4. 확인 버튼으로 생성 완료

2. 슬라이드 추가 및 편집

  1. 편집 모드로 전환
  2. 좌측 하단의 "+" 버튼으로 새 슬라이드 추가
  3. 텍스트 박스를 더블클릭하여 내용 입력
  4. 우측 속성 패널에서 폰트, 색상, 크기 조정
  5. 배경 이미지나 비디오 추가 (드래그 앤 드롭)

3. 화면 설정

메뉴 > 화면 > 화면 구성에서 출력 화면을 설정할 수 있습니다.

  • 청중 화면: 관객이 보는 메인 화면
  • 스테이지 화면: 발표자용 모니터
  • 해상도 및 위치: 각 화면의 해상도와 배치 설정

4. 미디어 관리

  1. 하단 미디어 빈에 파일 드래그 앤 드롭
  2. 지원 형식: MP4, MOV, JPG, PNG, MP3, WAV 등
  3. 미디어를 슬라이드로 드래그하여 배경으로 사용
  4. 미디어 재생 시간 및 반복 설정 가능
💡 유용한 팁:
  • 고해상도 이미지는 자동으로 최적화됩니다
  • 비디오는 루프 재생이 가능합니다
  • 미디어 폴더를 정리하여 빠른 검색이 가능합니다

⌨️필수 단축키

기본 조작

새 프레젠테이션 Cmd+N / Ctrl+N
저장하기 Cmd+S / Ctrl+S
실행 취소 Cmd+Z / Ctrl+Z
다시 실행 Cmd+Shift+Z / Ctrl+Y

화면 제어

메인 출력 ON/OFF Cmd+1 / Ctrl+1
스테이지 출력 ON/OFF Cmd+2 / Ctrl+2
미리보기 전환 Cmd+3 / Ctrl+3
화면 구성 Opt+Cmd+1 / Alt+Ctrl+1

편집 관련

슬라이드 복제 Cmd+Shift+D / Ctrl+Shift+D
편집기 열기 Cmd+E / Ctrl+E
슬라이드 분할 Alt+Enter
서식 없이 붙여넣기 Opt+Shift+Cmd+V / Alt+V

프레젠테이션 제어

다음 슬라이드 스페이스바 / 화살표
이전 슬라이드 백스페이스 / 화살표
화면 지우기 ESC
라이브러리로 이동 Cmd+L / Ctrl+L
⚠️ 중요: 라이브 프레젠테이션 중에는 Cmd+S / Ctrl+S로 자주 저장하여 작업 손실을 방지하세요!

💡전문가 팁

효율적인 워크플로우

1. 템플릿 활용

  • 자주 사용하는 디자인은 템플릿으로 저장
  • 일관된 브랜딩을 위해 색상 팔레트 설정
  • 폰트 조합을 미리 정의하여 시간 절약

2. 미디어 최적화

  • 비디오는 H.264 코덱 사용 권장
  • 이미지는 적절한 해상도로 압축
  • 오디오는 48kHz/24bit 또는 44.1kHz/16bit 사용

3. 라이브 운영

  • 리허설 시간을 충분히 확보
  • 백업 프레젠테이션 파일 준비
  • 네트워크 연결 상태 사전 점검
  • 원격 제어 앱을 활용한 무선 조작

성능 최적화

시스템 설정

  • 그래픽 카드: 전용 GPU 사용 권장
  • 메모리: 8GB 이상 RAM 권장
  • 저장공간: SSD 사용으로 로딩 속도 향상
  • 네트워크: 유선 연결로 안정성 확보

협업 및 백업

  1. 클라우드 동기화: 여러 컴퓨터 간 파일 공유
  2. 프로젝트 패키징: 관련 미디어 파일 포함하여 내보내기
  3. 정기 백업: 중요한 프레젠테이션은 별도 저장
  4. 버전 관리: 날짜별로 파일명 구분

🔧문제 해결

일반적인 문제들

화면 출력이 안 될 때

  1. 화면 구성에서 올바른 디스플레이 선택 확인
  2. Cmd+1 / Ctrl+1로 출력 활성화
  3. 케이블 연결 상태 점검
  4. 그래픽 드라이버 업데이트

프로그램이 느릴 때

  1. 사용하지 않는 미디어 파일 정리
  2. 캐시 파일 삭제 (Cmd+Ctrl+E / Ctrl+Shift+E)
  3. 해상도가 큰 이미지/비디오 최적화
